# Assets

Place your application icons in this directory:

- `icon.png` - General purpose icon (512x512 or larger recommended)
- `icon.ico` - Windows icon (for NSIS installer)
- `icon.icns` - MacOS icon (for DMG)

## Generating Icons

You can use tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [electron-icon-maker](https://www.npmjs.com/package/electron-icon-maker)
- Online converters

Or generate them manually using image editing software.
